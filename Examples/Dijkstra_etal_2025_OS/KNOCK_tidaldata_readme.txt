The file KNOCK_tidaldata2018 consists of tidal data from the tidal station near Knock in the Ems estuary from 01-01-2018 to 31-12-2018.
The station is maintained by the WSV and portions of the data can be periodically downloaded via www.pegelonline.wsv.de

The file contains the following columns:
Year #, Month #, Day of month, Time of day as fraction [0-1], Water level above reference in meters (German: m. Ã¼. NHN; meters above Amsterdam ordnance datum)
The data has a 5-minute resolution. Some missing data points may be interpolated.
